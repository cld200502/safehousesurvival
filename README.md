# 丧尸围城 2.5D - Godot 4 项目

## 快速开始
1. 安装 [Godot 4.3](https://godotengine.org/download)（推荐 4.3 LTS）
2. 用 Godot 打开 `project.godot` 文件
3. 按 F5 运行

## 操控方式

| 按键 | 功能 |
|---|---|
| WASD / 方向键 | 移动角色 |
| E / 空格 | 交互（搜柜子/开门/对话） |
| F / 鼠标左键 | QTE 校准判定 |
| I | 打开背包 |

## 游戏系统

### 核心循环：搜 → 打 → 撤
- **搜**：靠近柜子按 E，触发 QTE 校准搜物资
- **打**：碰到丧尸自动进入战斗，QTE 攻击丧尸
- **撤**：行动点用完或天黑后结束当天

### 数值系统
- **HP**：被丧尸攻击扣血，归零 = 死亡
- **饥饿**：随时间下降，归零 = 饿死（吃食物恢复）
- **理智**：随时间下降，归零 = 疯了（喝可乐恢复）
- **行动点**：每天 6 点，每个动作消耗 1 点
- **存活天数**：右上角显示

### 物品系统
| 物品 | 效果 |
|---|---|
| 🍖 食物 | 饥饿+30% |
| 🔪 小刀 | 武器，加成伤害 |
| 🥤 可乐 | 理智+20% |
| 💊 药品 | HP+40 |

### NPC & 队友系统
- 8 个可招募 NPC + 1 只猫
- 招募后增加战斗力，但消耗更多粮食
- 队友有**好感度**系统，太低会**叛变**
- 每天**随机事件**可能让队友**变异**

### QTE 校准难度等级
| 难度 | 转速 | 成功区域 | 适用场景 |
|---|---|---|---|
| easy (简单) | 150°/s | 60° | 初级搜刮 |
| normal (普通) | 220°/s | 45° | 标准交互/普通丧尸 |
| hard (困难) | 300°/s | 30° | 高级物资/精英丧尸 |
| extreme (极难) | 400°/s | 18° | Boss战/稀有事件 |

### 敲门事件
- 随机时间触发「咚咚咚——有人在敲门！」
- 选择**开门**：可能是幸存者（安全）也可能是丧尸（危险）
- 选择**不开门**：安全但错过机会

## 素材进度

把每张素材直接放到 `assets/textures/` 下即可：

```
assets/textures/
├── floor.png          # 地板贴图（深棕色水泥纹理） ✅已提供
├── wall1.png          # 墙面贴图（北墙/后墙，深棕褐色） ✅已提供
├── wall2.png          # 墙面贴图（侧墙，深灰褐色） ✅已提供
├── player_front.png   # 主角正面（支持多帧动画）
├── player_back.png    # 主角背面
├── cabinet.png        # 柜子（手绘风，带"柜"字）
├── door.png           # 门（手绘风 + 木纹，带"门"字）
├── zombie_front.png   # 丧尸（两只，只有正面）
├── npc_0~7.png        # 8个NPC (阿强/小美/老陈/林医生/眼镜妹/胖叔/小云/赵大爷)
├── npc_8.png          # 咪咪🐱猫咪
├── ui_panel.png       # UI状态面板底板 ✅已提供
├── qte_dial.png       # QTE圆盘表盘
├── dialog_bg.png      # 对话框横格纸底板
└── knock_icon.png     # 敲门事件图标
```

> 主角支持 12 帧动画（4方向×3帧），从大合辑自动拆分。
> 尺寸不用精确匹配。随便画就完事。

## 项目结构
```
project.godot          ← Godot 项目配置
scenes/main.tscn       ← 主场景
scripts/
├── game_systems/
│   ├── game_data.gd      ← 全局数据管理（状态、背包、队友、时间）
│   └── init.gd           ← 初始化（自动生成占位纹理、连接系统）
└── components/
    ├── player.gd         ← 角色移动控制
    ├── qte_system.gd     ← QTE 校准系统
    ├── ui_manager.gd     ← UI 管理（血条、背包、消息等）
    ├── dialogue_system.gd← 对话系统（参考幻象游园剧风格）
    ├── interactable_object.gd ← 可交互物体（柜子、门）
    ├── zombie_enemy.gd   ← 丧尸敌人 + 战斗
    └── npc_manager.gd    ← NPC 对话/招募/好感度
assets/textures/         ← 所有素材 PNG
```
